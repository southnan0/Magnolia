
```bash
# clone our repo
$ git clone https://github.com/southnan0/Magnolia.git

# change directory to your app
$ cd Magnolia

# install the dependencies with npm
$ npm install

# install the typings
$ npm i typings -g
$ typings install

# start the server
$ npm start
```
